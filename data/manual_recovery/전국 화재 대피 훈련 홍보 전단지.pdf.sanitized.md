# 화재 발생 시 국민 행동 요령 및 대피 가이드

본 문서는 화재 발생 시 국민이 안전하게 대피하고 초기 대응을 할 수 있도록 안내하는 행동 요령입니다.

## 1. 화재 발생 시 대피 방법

화재 발생 시 안전을 확보하고 신속하게 대피하기 위한 행동 요령입니다.

*   **경보 및 알림:** 불을 발견하면 "불이야!" 하고 큰소리로 외쳐 다른 사람에게 알리고 화재 경보 비상벨을 누릅니다.
*   **대피 경로:** 엘리베이터를 이용하지 말고 계단을 이용하여 대피하며, 아래층으로 대피할 수 없을 때는 옥상으로 대피합니다.
*   **연기 속 이동:** 불길 속을 통과할 때는 물을 적신 담요나 수건 등으로 몸과 얼굴을 감싸고, 한 손으로 코와 입을 젖은 수건 등으로 막은 후 다른 한 손으로 벽을 짚으며 낮은 자세로 이동합니다.
*   **문 확인:** 문손잡이를 만져보았을 때 뜨겁지 않으면 문을 조심스럽게 열고 밖으로 나갑니다.
*   **출구 봉쇄:** 출구가 막혔을 경우, 연기가 방안으로 들어오지 못하도록 물을 적셔 문틈을 옷이나 이불로 막고 구조를 기다립니다.

## 2. 소화기 사용 요령

화재 초기 진압을 위한 소화기 사용 방법입니다.

*   **사용 준비:** 소화기를 불이 난 곳으로 옮긴 후 손잡이 부분의 안전핀을 뽑습니다.
*   **방향 설정:** 바람을 등지고 서서 호스를 불이 난 쪽으로 향하게 합니다.
*   **소화:** 손잡이를 힘껏 움켜쥐고 빗자루로 쓸듯이 불을 뿌립니다.
*   **보관:** 소화기는 잘 보이고 사용하기 편리한 곳에 두되, 햇빛이나 습기에 노출되지 않도록 보관합니다.

## 3. 화재 발생 시 119 신고 요령

신고 시 필요한 정확하고 명료한 정보 전달 방법입니다.

*   **신고 내용:** 소방서에 신고할 때 불이 난 내용(예: 저희 집 주방에 불이 났어요. 2층 집입니다)을 간단하고 명료하게 설명합니다.
*   **주소 전달:** 주소를 정확하게 전달합니다.
*   **연락 유지:** 소방서에서 알았다고 할 때까지 전화를 끊지 않습니다.
*   **긴급 신고:** 휴대전화의 경우, 사용 제한되거나 개통이 안 된 전화도 긴급신고가 가능합니다.

## 4. 고층 건물 화재 발생 시 대피 요령

고층 건물에서 화재가 발생했을 때의 특별 대피 지침입니다.

*   **문 처리:** 화재 발생한 곳에서 탈출할 때는 문을 반드시 닫고 나와야 하며, 열린 문이 있다면 모두 닫습니다.
*   **문 열기:** 닫힌 문을 열 때는 손등으로 문의 온도를 확인하고, 뜨거우면 절대로 열지 말고 다른 비상통로를 이용합니다.
*   **대피 실패 시:** 건물 밖으로 대피하지 못한 경우에는 밖으로 통하는 창문이 있는 방으로 들어가서 구조를 기다립니다.
*   **연기 차단:** 방안으로 연기가 들어오지 못하도록 문틈을 커튼 등으로 막고, 주위에 물이 있다면 옷에 물을 적셔 입과 코를 막고 숨깁니다.
*   **위치 알림:** 통신 수단이 있다면 전화하여 자신의 위치를 정확하게 알립니다.
