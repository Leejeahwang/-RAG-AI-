# 🎥 Edge Saver - Phase 3: 실시간 CCTV 감시 파이프라인으로의 진화 계획

현재의 `camera.py`와 `fire_detector.py` 코드는 PC 환경에서 안전하게 기능이 동작하는지 수동 캡처(키보드 누름)로 검증하기 위한 테스트(Phase 2)용으로 완성되었습니다. 
하지만 시스템의 최종 완성 형태인 **"라즈베리파이(Edge 디바이스) 무인 24시간 감시망"**으로 올리기 위해, 향후 기존 코드를 어떻게 재활용하고 조립할 것인지에 대한 설계안(Plan)을 정리합니다. (핵심 AI 감지 로직은 건드리지 않고 그대로 씁니다!)

---

## 1. 📷 카메라 입력의 백그라운드 무인화 (Headless Streaming) [✅ 완료]
- **현재 구조의 한계:** 모니터 화면에 그림(`cv2.imshow`)을 띄우고 사용자가 직접 키보드('c')를 누를 때까지 코드가 멈춰 있습니다(Blocking).
- **진화 계획 (적용 됨):** 
  - `cv2.imshow` 코드 블록을 완전 제거하여, 모니터가 아예 없는 라즈베리파이 현장 설치 환경에서도 백그라운드 시스템 서비스 상태로 구동되게 바꿉니다.
  - 최신 영상을 계속해서 버퍼(메모리)에만 읽어 들이고 덮어쓰게 하여 영상 지연을 방지합니다. (`cctv_service.py` 데몬 스레드로 작동 중)

---

## 2. ⚡ 라즈베리파이 발열/CPU 효율화 전략 (Inference 최적화)
가장 중요한 부분입니다. PC와 달리 라즈베리파이는 GPU 성능이 낮아 CCTV처럼 초당 30장씩 들어오는 모든 영상을 AI로 분석시키면 5분 만에 발열로 다운(Throttling)됩니다. 따라서 똑똑하게 AI를 껐다 켰다 하는 스위칭 로직이 도입되어야 합니다.

- **[전략 A] 멀티센서 퓨전 트리거 (강력 추천):**
  - 평소에는 카메라가 촬영만 하고 AI 모델(뇌)은 휴면(대기) 상태를 유지합니다.
  - 재황 님이 구축할 하드웨어 센서(`smoke.py`, `temperature.py`)에서 일차적인 연기/가스/온도 상승을 감지하면 인터럽트 신호를 보냅니다.
  - 신호가 떨어지는 찰나의 순간에만 **Vision AI가 탁 깨어나 가장 최신 프레임을 가져다 화재(`fire_detector.py`)인지 2차 검열**을 수행합니다. 
  - 이를 통해 거짓 오류(오경보)를 원천 차단하면서도, 장비 체력을 100% 보존할 수 있습니다.

- **[전략 B] 쿨타임(주기적) 스캔 모드 (✅ 현재 cctv_service.py 적용 됨):**
  - 센서 망이 닿지 않는 물리적으로 멀리 떨어진 장소를 카메라 단독으로 커버할 때 씁니다.
  - 30장의 프레임을 다 분석하지 않고 **매 3~5초마다 단 1장(0.2 FPS)만 쏙 빼내어** 오프라인 YOLOv8 모델로 던져 검사합니다.

---

## 3. 🧠 라파 전용 모델 고속화(양자화) 포맷 변환 [✅ 완료]
- 원본 `fire_smoke.pt` 파일은 부동소수점(FP32) 연산 오버헤드가 있어 라즈베리파이(ARM 아키텍처)에서는 발열과 프레임 드랍을 유발합니다.
- **적용 완료 (OpenVINO INT8):** `ultralytics` 및 `nncf` 캘리브레이션을 통해 원본 `.pt` 모델(6MB)을 **Intel OpenVINO(INT8)** 포맷으로 극한의 양자화(Quantization) 변환을 완료하였습니다. 
- **성능 개선 결과:** 용량이 절반에 가까운 **3.4MB**로 압축(-43%)되었으며, 캘리브레이션 튜닝 효과로 화재 감지 신뢰도(Confidence)가 기존 34%에서 **58.5%**로 대폭 상승하는 놀라운 시너지를 달성했습니다. (자세한 내용은 `benchmark_results.md` 참조)

---

## 4. 🚨 최종 통합 루프 (main.py 통합 시나리오)
현재 모듈별로 파편화된 코드들을 하나의 거대 파이프라인으로 연결합니다.

1. **[센서망 이상 신호] ➡️ [Vision AI 호출]**
2. **[Vision AI]** "화재 확신도 91% (FIRE 감지)" 판정! (`fire_detected == True`)
3. **[Alerts 모듈 방아쇠]** 즉시 삐- 사이렌을 울리고 LED를 점멸시킵니다.
4. **[RAG / LLM 모듈 호출]** AI가 센서 온도를 기반으로 소방 매뉴얼을 0.5초 만에 읽어들입니다.
5. **[Voice 음성 송출]** RAG가 만든 "현재 시스템실 화재, 전원을 차단하고 C구역 문으로 대피하세요" 지침 텍스트를 `tts.py`가 스피커로 또박또박 방송합니다.
6. 관제 대시보드(GUI)에는 화재를 잡아낸 불꽃 사진 한 장이 박제되어 전송됩니다.
