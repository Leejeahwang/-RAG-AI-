🖥️ Edge Saver 통합 관제 대시보드 (Dashboard)
본 모듈은 Edge Saver 프로젝트의 프론트엔드이자 통합 제어 시스템으로, 라즈베리파이 5 환경에서 실시간 시각화 및 AI 음성 대응을 총괄합니다.

📐 인터페이스 레이아웃 (Layout)
사용자 편의성과 긴급 상황 시 가독성을 최우선으로 고려하여 3단 분할 레이아웃을 적용했습니다.

좌측 (Main Panel): 실시간 CCTV 스트리밍 및 YOLOv8 기반 화재/연기 감지 바운딩 박스 출력.

우측 상단 (Sensor Metrics): 온도(°C), 연기 농도, 가스 농도를 실시간 수치(st.metric)로 표시.

우측 하단 (Risk Gauge): fusion.py의 연산 결과(Level 1~5)를 동적 막대바(Progress Bar) 형식으로 시각화.

하단 (Tactical AI Intelligence): AI 사령관의 대응 지침 출력 및 음성 인식 상태 표시 전용 영역.

🚨 핵심 전술 로직 (Emergency Logic)
1. 단계별 시각 경보 (Visual Alert)
Level 1~3: 안정적인 관제를 위한 표준 테마 유지.

Level 4~5 (긴급): 전체 화면 배경 빨간색 고주기 점멸(rapid_blink) 효과 강제 주입. 관제사의 즉각적인 주의 환기.

2. 선제적 AI 개입 및 연속 음성 루프 (Continuous Voice AI)
긴급 상황 시 마우스 조작 없이 오직 음성만으로 매뉴얼을 검색할 수 있는 '핸즈프리' 모드를 지원합니다.

선제 가이드: 4단계 감지 즉시 AI가 먼저 "긴급 상황입니다. 대응 매뉴얼에 대해 질문하십시오"라고 음성 출력.

재귀적 질의 루프: 관제사의 질문 수렴 → RAG 엔진 검색 → TTS 답변 출력 → 즉시 재경청 상태 진입.

동기화 처리: AI 답변 중에는 마이크를 닫아 에코 현상을 방지하며, 답변 종료 후 st.rerun()을 통해 최신 센서 데이터 업데이트와 동시에 다음 질문을 대기합니다.

🛠️ 실행 방법 (Usage)
# 최상위 디렉토리에서 실행
streamlit run dashboard.py
