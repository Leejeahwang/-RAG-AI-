# 🧠 Edge Saver: RAG Intelligence Engine

이 폴더는 '엣지 세이버(지능형 화재 감시 시스템)'의 핵심 두뇌인 **RAG(검색 증강 생성) 엔진** 모듈을 담고 있습니다. 라즈베리파이의 제한된 자원에서도 정확하고 빠른 재난 대응 지침을 생성하도록 최적화되었습니다.

---

## 📂 주요 모듈 구성 (3인방)

| 파일명 | 역할 | 핵심 기능 |
| :--- | :--- | :--- |
| **`loader.py`** | 지식 손질기 | PDF/TXT 매뉴얼을 읽어 AI가 처리하기 쉬운 크기(Chunk)로 분할 및 로깅 |
| **`retriever.py`** | 지능형 사서 | 지식 조각을 벡터 DB(Chroma)에 저장하고 상황에 맞는 지침을 초고속 검색 |
| **`chain.py`** | 현장 사령관 | 검색된 지식과 상황 맥락을 결합하여 '5대 강령' 기반 최종 지시 생성 |

---

## 🚨 핵심 설계 원칙 (5대 행동 강령)

본 엔진은 재난 상황에서 AI의 오판을 방지하고 인명을 보호하기 위해 아래 **5대 절대 원칙**을 강제하도록 설계되었습니다.

1. **지체 없는 지침 제공**: 복잡한 설명 생략, 즉각적인 대응 방안 우선 출력.
2. **구체적 정보 안내**: 위험물질 종류, 소화기 위치, 최단 대피 경로 등 실질 데이터 포함.
3. **명령조의 한국어**: "~입니다" 식의 평서문 지양, "~하십시오" 등의 단호한 명령조 사용.
4. **추측 금지 (Fact-Only)**: 매뉴얼에 없는 내용은 절대로 지어내지 않음 (근거 중심).
5. **인명 안전 최우선**: 모든 판단과 지시의 최우선 순위는 생존 및 구조에 집중.
